欢迎来到小鸡庄园 ！

小鸡庄园是一款文字游戏，由“XJZY开发组”开发和测试 图1-1 2.8运行截图

项目时间点

V1

V1版本于2022年3月19日开始开发，整个V1版本于同年的3月21日19点48分结束开发，于2022年3月29日正式发布，一共有1.0~1.6共7个版本。目前1.5版本可以在这里下载

V2

V2版本目前共有12个版本，其中包括3个（2.0.1、2.0、2.3（BUG））重大BUG修复版本和9个普通版本

可应用平台

只要你安装了Python，在任何平台都可以运行、编译小鸡庄园（用安卓玩可能会出现奇奇怪怪的BUG） 图2-2 安卓游戏截图

所需运行文件

使用Windows方法二和MacOS需要安装Python3.6及以上（Linux自带）和Python中的colorama库，我们比较推荐安装3.8版本的Python Python：https://python.org/downloads/
colorama：pip install colorama（安装Python后复制到cmd执行）

其他事项（1、2必看！）

  1. 运行时会自动创建XJZY.db，此为数据库文件，用来存储数据。删除后会自动重建，但只有2.5版本及以上可以恢复存档

  2. 比较推荐的入门版本为 2.2，功能齐全，txt文本储存，还有彩色字体，拯救眼睛！

  3. 你可以在 文件交换仓库 找到早期的简单代码文件等奇奇怪怪的代码和开发组趣事

版本命名准则

正式版

在大部分情况下应只有一份应用程序及及一份Python源文件,格式为:XJZY-X.X

公测版

每一个正式版本的公测版应全部在同一个文件夹下(发行时分开),格式为:XJXYx.xβ/XJXY-X.XBeta,简写(标签命名)为x.xb

内测版

内测版版本非常多，一般仅供开发组成员测试使用,命名格式为:XJXYx.xα/XJXY-X.XAlpha

注:标签命名应使用全数字命名，例如：2.3（bug）标签应命名为2.3.1（同时作为内部版本名），以防止系统排版错误
